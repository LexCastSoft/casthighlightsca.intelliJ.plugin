Version 1.2

Description

Highlight extension for visual studio code: Allow the user to get vulenrability informations about its installed dependency.

Features

• Connection to Highlight API
• Retrieving dependency from pom.xml and package.json files
• Displaying each dependency vulnerability list for the current version
• Displaying global counter of vulneraibility for the whole dependency

Requirements

You need to have an Highlight account to get connected to this extension (API connection required)
